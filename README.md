# Healthcare Cost Prediction System

## Overview

A Machine Learning application that predicts annual healthcare costs based on patient demographics, lifestyle habits, medical conditions, and insurance information.

## Dataset

- 5,000 patient records
- 20 features
- Target Variable: annual_medical_cost

## Workflow

1. Data Loading
2. Data Understanding
3. Data Cleaning
4. Feature & Target Separation
5. Train-Test Split
6. Feature Engineering (Encoding + Scaling)
7. KNN Model Training
8. Model Evaluation
9. Model Saving
10. Streamlit Deployment

## Model Performance

- KNN Regressor
- R² Score: 74%
